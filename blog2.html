TOP 10 MOST ANTICIPATED GAMES OF 2018

10. Spider-Man

Spider-Man is an upcoming action-adventure game developed by Insomniac Games and published by Sony Interactive Entertainment. The game is planned to release on Q1/Q2 of 2018. The story of this game is not connected or tied to movie or comics. Game is set in an open world New York and can be played from third person perspective. And also we can play as both Spiderman and Peter Park. This game will be exclusive for PS4

9. Scum

Scum is a multiplayer online survival game developed by Gamepires and published by Devlover Digital. This Game is planned to enter steam as early access in Q1 of 2018. The game is developed in Unreal Engine 4. After watching gameplay footages this game looks amazing and promising. Lot of people are hyped for this game and some may think its an early access game and it will stay forever in early access.

8. A Way Out

A Way Out is an upcoming action-adventure multiplayer game developed by Hazelight Studios and published by Electronic Arts (EA). The Game is scheduled to release on March 23, 2018 for PC, PS4 and Xbox One. This game can be played in either online or local split-screen coop between two players. It will not feature single-player mode. And it can be played from third person perspective.

7. State Of Decay 2

State of Decay 2 is an upcoming zombie survival video game developed by Undead Labs and published by Microsoft Studios. It is a sequel to State of Decay which was released in 2013. Game is set in an open world and can be played from third person perspective. And this time game will feature coop mode up to 3 players.Release date has been not announced but it will be releasing for PC and Xbox One.

6. Anthem

Anthem is an upcoming online multiplayer action role-playing video game developed by BioWare and published by Electronic Arts (EA). It is planned to release on Q4 of 2018 for PC,PS4 and Xbox One. Game is set in an open world and can be played from third person perspective. And it will feature both the single player and multiplayer mode.

5. Days Gone

Days Gone is an upcoming action-adventure survival horror video game developed by SIE Bend Studio and published by Sony Interactive Entertainment. The game is developed using Unreal Engine 4. And is set to release in 2018. Player takes control of the protagonist Deacon St. John and can be played from third person perspective. The game is set in a post-apocalyptic open world.

4. Metro Exodus

Metro Exodus is an upcoming first-person shooter game developed by 4A Games and published by Deep Silver. It is the third installment of Metro Games which is based on novel. It is developed using 4A Engine and set to release in Q4 of 2018 for PC,PS4 and Xbox One. Game is set in a post-apocalyptic wasteland of Russian Federation. And it will feature only single player mode like previous games in the series.

3. Far Cry 5

Far Cry 5 is an upcoming action-adventure first-person shooter video game developed and published by Ubisoft. And it is set to release on March 27, 2018 for PC,Ps4 and Xbox One. It will be the latest entry to the famous franchise Far Cry. This game can be played individually or with another player in a multiplayer co-op mode. And it will also feature Map editor like previous games but with some new features.

2. God Of War

God of war is an upcoming action-adventure video game developed by SIE Santa Monica Studio and published by Sony Interactive Entertainment. It is the sequel to the god of war 3 and it is a soft reboot of franchise. And it is set to be released in Q1 of 2018 for PS4. In previous games they were based on Greek Mythology but this time they are based on Norse Mythology. Series protagonist Kratos will return with his son.

1. Red Dead Redemption 2

Red Dead Redemption 2 is an upcoming action-adventure Western-themed game which developed and published by Rockstar Games. And it is scheduled to be released in Q2 of 2018 for PS4 and Xbox One. This game will feature both single-player and multiplayer mode and it will be the prequel to the Red Dead Redemption which was released in 2010. The player controls An outlaw Arthur Morgan from third person perspective.
